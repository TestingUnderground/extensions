About

Literally nobody (yes, as opposed to figuratively nobody) asked me to create this site, but I did. It hosts Hacknet Extensions in a place other than the Workshop, or Nexus Mods. More specifically, it hosts them on GitHub.

This is completely legal, as I'm getting the files using Valve's SteamCMD, and moreover Hacknet's developer, Matt Trobbiani, has (thankfully) allowed anonymous workshop downloads for people that use the GOG/Itch/? versions.


If I receive a request from Matt to take the site down, I will comply.
If you do not want to use my site, you can use the following guide: Downloading Steam Workshop Mods
